# Criterios de votacion y carga editorial

## Principio general

La votacion se realiza fuera del sistema, en publicaciones oficiales de Instagram. El sitio funciona como repositorio editorial de resultados, tablas y fixture.

## Datos que debe registrar la administracion

Por cada partido:

- participante 1
- participante 2
- votos del participante 1
- votos del participante 2
- URL del post de Instagram
- estado del partido
- ganador, si corresponde
- notas internas o aclaraciones

## Reglas para cierre de resultado

- solo se debe publicar resultado final cuando la votacion este cerrada
- si el partido sigue abierto, no debe presentarse como definitivo
- si hubiera empate, la administracion podra definir ganador manual si la fase lo exige

## Reglas para consistencia editorial

- usar la misma captura o conteo base para registrar ambos valores
- evitar cambios posteriores salvo correccion fundada
- dejar nota interna si hubo ajuste manual
- verificar que la URL del post corresponda al partido correcto

## Transparencia recomendada

- publicar en el sitio el estado del partido
- enlazar el post original cuando exista
- mantener criterio uniforme de carga en toda la copa

## Criterio de desempate manual

Si en una fase eliminatoria el conteo final quedara empatado y fuera necesario un clasificado:

- registrar el empate en votos
- consignar el ganador manual definido por la administracion
- dejar aclaracion en notas o texto complementario
