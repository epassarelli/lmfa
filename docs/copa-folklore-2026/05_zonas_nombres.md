# Nombres sugeridos para zonas

## Criterio

Los nombres de zonas pueden tomar referencias amplias del folklore argentino sin asociarse a marcas externas ni replicar formatos oficiales de competiciones deportivas.

## Opcion 1: regiones y paisajes culturales

- Zona Norte
- Zona Litoral
- Zona Centro
- Zona Cuyo
- Zona Patagonia
- Zona Monte
- Zona Quebrada
- Zona Llanura

## Opcion 2: conceptos editoriales

- Zona Raices
- Zona Encuentro
- Zona Tradicion
- Zona Guitarra
- Zona Copla
- Zona Festival
- Zona Camino
- Zona Fogon

## Opcion 3: sedes simbolicas sin apropiacion institucional

- Zona Cosquin
- Zona Salta
- Zona Santiago
- Zona Tucuman
- Zona Mendoza
- Zona Corrientes
- Zona Cordoba
- Zona Jujuy

## Recomendacion

Para el MVP conviene iniciar con nombres simples y claros. La opcion recomendada es:

- Zona A
- Zona B
- Zona C
- Zona D
- Zona E
- Zona F
- Zona G
- Zona H

Esto evita ambiguedad y deja abierta una personalizacion editorial futura.
