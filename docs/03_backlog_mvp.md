# Backlog MVP

## 1. Épica 1 - Gestión de publicadores y organizaciones
### HU1.1 Registro de publicador
Como usuario, quiero registrarme para poder cargar contenido.

### HU1.2 Selección de perfil
Como usuario, quiero indicar si soy artista, productora, peña, festival u organizador.

### HU1.3 Alta de organización
Como publicador, quiero crear mi organización para asociar contenidos.

### HU1.4 Gestión de miembros
Como dueño de organización, quiero invitar colaboradores.

### HU1.5 Verificación
Como admin, quiero marcar una organización como verificada.
